  2. Restart terminal/computer
  3. Verify: npm --version

  2. Allow camera/microphone access
  3. Try another browser (Chrome recommended)
  4. Restart browser

  2. Check camera is working in browser
  3. Model loads OK if backend shows "âœ… Model loaded"
  4. Try test image first
